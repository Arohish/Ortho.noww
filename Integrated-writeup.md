The OrthoNow landing page was developed as a responsive healthcare website focused on generating consultation leads. The project integrates a consultation booking form with client-side validation to ensure users enter valid details before submission.
Google Tag Manager (GTM) event tracking was implemented using the dataLayer. When a user submits the consultation form, a custom event (consultation_form_submitted) is pushed to the dataLayer along with parameters such as patient name, phone number, selected clinic, lead source, and page name. This enables marketing and analytics teams to track lead generation efficiently.
The website also includes interactive features such as smooth scrolling, FAQ accordion, animated counters, scroll reveal effects, a back-to-top button, WhatsApp contact button, and a success message after form submission, providing an improved user experience.
Performance optimization was carried out using Google PageSpeed Insights. The website achieved a Performance score of 93, Best Practices score of 100, and SEO score of 100, demonstrating good optimization, responsive design, and adherence to modern web development standards.
The project is deployed using GitHub Pages, with the complete source code maintained in a GitHub repository for version control and easy collaboration.
